# Plugin-List

The plugin syntax is as follows:

    {exp:plugin_list backspace="6"}

        {plugin_name}<br />

    {/exp:plugin_list}

Parameters:

- `backspace`
- `which` - chooses a plugin folder (main | thirdparty | both)
